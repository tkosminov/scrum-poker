# Scrum-Poker Front

* Vue 3

## Dependencies

* NodeJS 20
  * [ubuntu/debian](https://computingforgeeks.com/how-to-install-node-js-on-ubuntu-debian/)

## Installation

```bash
npm ci
```

## Running the app

```bash
npm run start:dev
```

app listening on `http://localhost:5173`

## Docker

```bash
sudo docker-compose up --build
```

app listening on `http://localhost:5173`
